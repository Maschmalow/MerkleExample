Class Hierarchy

This inheritance list is sorted roughly, but not completely, alphabetically:

  • class merkle::Hashable Inherit this class and implement addToHash() to define hashing of a custom class.
  • class merkle::Hasher SHA256 instance.
  • class merkle::MerkleMembershipProof Class holding a membership proof.
  • class merkle::Hash byte array containing a hash digest.
  • struct merkle::MerkleMembershipProof::NodeHash
  • interface contiguous_hashable Concept modelling continuous arrays of directly hashable elements, ie, types that can be hashed with a single call.
  • interface derived_hashable Concept modelling types supporting hashing though addToHash().
  • interface directlyHashable Concept modelling types that can be hashed as is, using sifzeof.
  • interface implementsAddToHash Concept modelling types implementing addToHash().
  • interface iterable Concept modelling types that supports Range-based for loop.
