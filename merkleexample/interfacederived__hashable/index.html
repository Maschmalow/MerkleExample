Concept modelling types supporting hashing though addToHash(). More...

Detailed Description

We do not use the implementsAddToHash directly because we would risk having external classes inadvertently implementing addToHash in unexpected ways. Other implementations are possible with different trade-offs, eg, CRTP, type tag, virtual class, etc.
